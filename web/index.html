TactileText

An interaction-focused reading experience exploring hover, click focus, and semantic language support.

View Source Explore Features

What It Shows

  • Offline Demo Mode

    Runs out of the box with bundled reading content and prepared translations.

  • Interaction Design

    Focused on tactile reading interactions: hover, click, and scroll depth control.

  • Optional AI Layer

    Real AI features can be enabled in Xcode with GEMINI_API_KEY.

  • Clean Architecture

    Designed to keep core experience usable even without credentials.

Run On Mac

  1. Open TactileText.xcodeproj in Xcode on macOS.
  2. Select the TactileText scheme.
  3. Prefer My Mac (Designed for iPad) when available on Apple Silicon.
  4. If that target is unavailable, use a fixed iPad profile in Simulator for consistent demos.
